# Button

Button component with different props.

#### Example

<Canvas of={Button.Primary} />

## Usage

```ts
import { Button } from "@umi-digital/ui";

const Example = () => {
  return (
      <Button type="default">Click Me</Button>
  )
};

export default Example;
```

#### Arguments

- **text** _`string`_ - A string that represents the text content of the button.
- **type** _`enum(string)`_ - An enum of strings indicating whether the button should have a primary styling or not. Typically, a primary button stands out as the main action in a user interface.
- **link** _`string`_ - A string that represents the link content of the button.
